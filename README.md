# custom-webpack

手写 webpack 、 plugin 、 loader 、 babel-plugin

### 安装依赖

```
npm install
  or
yarn install
```

## 测试运行

### webpack

```cmd
  npm run myWebpack
  // 生成打包文件index.js至myDist目录后，可以使用html文件引入该脚本使用
```

### plugin 和 loader 和 babel-plugin

```cmd
  npm run build
  // 可以查看 webpack.config.js 内部的webpack插件和loader和babel插件及对应的文件的编写
```
